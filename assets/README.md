# 资源文件目录规范

| 目录       | 用途                 |
|------------|----------------------|
| /icons     | 扩展图标             |
| /styles    | CSS样式文件         |

## 文件命名规范
- 图标文件：`icon-{size}-{color}.svg`
- CSS文件：`{component}.module.css`

# 测试目录规范
| 目录            | 用途                 |
|-----------------|----------------------|
| /tests/unit     | 单元测试             |
| /tests/integration | 集成测试         |
